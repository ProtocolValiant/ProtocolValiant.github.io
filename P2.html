Genshin Impact has a total of 7 known elements each corrosponding to the 7 archons and their respective regions. Those elements each have their own unique elemental reactions when combined with one another. These 7 elements are Anemo, Geo, Electro, Dendro, Hydro, Pyro and Cryo. Every playable character in game has control over one of the seven elements and each characters have their own different roles and utilities which the player can use in game.
To the left is a diagram of current elemental reactions as of version 2.8. Note that Dendro has only one reaction currently in game as of version 2.8 as no Dendro character has yet been implemented into the game until the region of Sumeru comes out, which is speculated to be in update version 3.0
